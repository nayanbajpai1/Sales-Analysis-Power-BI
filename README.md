# Sales Analysis Power BI Project

## Project Overview
This project is a **Power BI sales analysis report** designed to provide insights into sales performance across various dimensions, including revenue, profit, orders, and quantity. The report includes interactive visuals, comparison analysis, and ranking insights to help stakeholders make informed decisions.

## Pages in the Report
1. **Sales Overview**  
   - High-level KPIs: Revenue, Order Quantity, Profit  
   - Trend analysis: Revenue by Month/Year (bar & line charts)  
   - Geographic insights: Revenue by State (map visual)
